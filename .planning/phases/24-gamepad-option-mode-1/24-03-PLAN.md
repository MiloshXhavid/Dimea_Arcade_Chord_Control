<task type="checkpoint:human-verify" gate="blocking">
  <what-built>
    Phase 24: Gamepad Option Mode 1 — all new face-button arp dispatch added, plus per-mode
    control highlight.

    Changes in this phase:
    - GamepadInput: 4 new atomic signals + Mode 1 face-button detection (btnMode1* ButtonState trackers)
    - PluginProcessor: Mode 1 face-button arp dispatch block; looper consume gated to optMode != 1
    - PluginEditor: Mode 1 label changed from "OCTAVE" to "ARP"; per-mode control highlight in timerCallback

    Requirements to verify: OPT1-01 through OPT1-07
  </what-built>
  <how-to-verify>
    Setup: Load ChordJoystick in Ableton Live on a MIDI track. Connect PS controller.
    Start with looper stopped. Open MIDI Monitor or watch plugin UI throughout.

    **PREP: Enter Mode 1**
    - Press Options button once. Plugin UI label should show "ARP" in green (not "OCTAVE").
    - The arp section controls (Arp toggle, Rate, Order, RND Sync) and the four octave knobs
      should show a subtle green tint/highlight.
    - All other controls should look normal (no tint).
    - If label still shows "OCTAVE" or no highlight appears, the UI changes did not deploy — abort and redeploy.

    **OPT1-01: Circle toggles arp + looper start**
    1. Confirm looper is STOPPED (not playing).
    2. Press Circle. Arp Enabled button in UI should turn on. Looper should start playing from beat 0.
       In DAW MIDI monitor: single arp notes should begin firing at configured rate.
    3. Press Circle again. Arp should turn off. Notes stop immediately.
    4. With looper already PLAYING (start it manually), press Circle once more.
       Arp turns on — looper does NOT reset position (no jump to beat 0).
    5. With DAW sync active: press Circle. Arp arms and waits for next bar boundary before first note.

    **OPT1-02: Triangle steps Arp Rate**
    1. Note the current Rate value in the UI (e.g., 1/8).
    2. Press Triangle once. Rate should advance one step (e.g., 1/8 → 1/16T or next option).
       Full sequence: 1/4 → 1/8T → 1/8 → 1/16T → 1/16 → 1/32 → back to 1/4.
    3. Press Triangle twice quickly (within ~300ms). Rate should step backward one position.
    4. Verify the Rate dropdown in the UI updates to match.

    **OPT1-03: Square steps Arp Order**
    1. Note the current Order value in the UI (e.g., Up).
    2. Press Square once. Order should advance one step.
       Full sequence: Up → Down → Up+Down → Down+Up → Outer-In → Inner-Out → Random → back to Up.
    3. Press Square twice quickly. Order should step backward one position.
    4. Verify the Order dropdown in the UI updates.

    **OPT1-04: X button toggles RND Sync**
    1. Note RND Sync state in the UI (the randomClockSync toggle).
    2. Press X (Cross). RND Sync should toggle. Press again — toggles back.
    3. Verify the UI reflects each change.

    **OPT1-05: Pad+R3 still toggles Sub Oct (regression check)**
    1. Hold L1 (Root pad) + press R3. Sub Oct for Voice 0 should toggle on/off in the UI.
    2. Test at least one other voice combination.

    **OPT1-06: R3 alone = no-op (regression check)**
    1. Make sure no pad buttons are held.
    2. Press R3 alone. Nothing should happen — no MIDI panic, no all-notes-off in monitor.

    **OPT1-07: Mode 2 INTRVL regression check**
    1. Press Options button again to enter Mode 2. Label should show "INTRVL" in red/orange.
    2. The globalTranspose and interval knobs should now show the Mode 2 highlight tint (muted blue).
       Arp/octave controls should return to no-highlight (their Mode 1 tint should be gone).
    3. Press D-pad Up — globalTranspose should step. Press D-pad Down — thirdInterval should step.
       Confirm both in UI.
    4. Press Circle/Triangle/Square/X in Mode 2 — looper face buttons should still fire
       (looper starts/stops/records/deletes normally in Mode 2 since looper dispatch runs when optMode != 1).

    **Mode 0 regression check (unlisted but important)**
    1. Press Options button again to return to Mode 0. Label should show "OPTION".
    2. ALL highlight tints should be gone — every control back to its normal colour.
    3. Press Circle — looper delete should fire. Press Cross (X) — looper start/stop.
       Confirm looper still responds to face buttons in Mode 0.
  </how-to-verify>
