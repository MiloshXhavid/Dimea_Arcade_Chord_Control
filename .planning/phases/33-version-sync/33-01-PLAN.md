```
gh release create v1.7 \
  "C:/Users/Milosh Xhavid/get-shit-done/installer/Output/DimeaArcade-ChordControl-v1.7.0-Setup.exe" \
  --title "DIMEA PLUGIN - CHORD JOYSTICK MK2 v1.7" \
  --notes "$(cat <<'NOTES'
v1.7 — Space Joystick

The biggest visual and feel update yet. v1.7 brings the joystick to life with a physics-based cursor, a full space-themed UI overhaul, expanded gamepad routing, and a batch of important bug fixes including triplet subdivisions.

**Visual Overhaul**
- Space background with milky way particle field (zero per-frame allocation, baked in resized())
- BPM-synced glow ring on the joystick pad — one full pulse cycle per beat at any tempo
- Semitone grid overlay: in-scale vs out-of-scale cells differentiated by alpha

**Joystick Physics**
- Spring-damper cursor with inertia — smooth settle after release, kDamping=0.90 (~150ms)
- Perimeter arc indicator shows cursor position around the joystick boundary
- Note-label compass at cardinal positions (root, third, fifth, tension labels at N/S/E/W)

**Gamepad SWAP/INV**
- SWAP: routes left stick to pitch + JoystickPad cursor, right stick to filter CC
- INV: swaps X and Y axes on both sticks (90° rotation, not sign inversion)
- Both modes save with preset via APVTS bool params; gamepad UI shows cyan/amber lit indicators
- L3/R3 labels update dynamically (MOD WHL / PITCH) based on SWAP state

**Bug Fixes & Triplets**
- Arp step counting fixed: start-of-block index prevents skipped beats and DAW loop/rewind hangs
- Gate length now applies to joystick triggers (not just random/arp sources)
- CC mode expansion: CC74/CC71/CC12/CC76 available on both X and Y left-stick axes (indices updated — existing presets with LFO modes 2-5 will remap)
- Left-stick deadzone: smooth rescaling from boundary (no abrupt jump); stick-in-deadzone returns CC to MOD FIX base
- Looper reset now syncs random free phases (randomPhase_[v]=0, prevSubdivIndex_[v]=-1)
- Population knob: deterministic subdivision shift (default 32 = no change; ±7 subdivison steps at extremes)
- Triplet subdivisions (1/1T–1/32T) added to random trigger and quantize subdivision selectors
NOTES
)" \
  --latest \
  --repo MiloshXhavid/Dima_Plugin_Chrdmachine
```

The `--latest` flag makes this the new Latest release on GitHub, superseding v1.5.
v1.6 was never published to GitHub — v1.7 is the successor to v1.5 as the public Latest.

If gh CLI is not authenticated, run `gh auth login` first and follow prompts.
