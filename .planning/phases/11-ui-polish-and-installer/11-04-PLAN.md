  <how-to-verify>
**Step 1 — Build the plugin:**
```
cmake --build build --config Release
```
Confirm build succeeds with zero errors.

**Step 2 — Open plugin in DAW (Ableton/REAPER):**
Load the plugin on a MIDI track and open the editor.

**Step 3 — Verify section panels (UI-01):**
- Confirm the LOOPER section (left column, transport buttons area) has a rounded rectangle border with "LOOPER" text floating centered on the top border line
- Confirm the FILTER MOD section (right column) has a rounded rectangle border with "FILTER MOD" text on the top border
- Confirm the GAMEPAD section (right column, GAMEPAD ON/OFF + PANIC! + status label row) has a rounded rectangle border with "GAMEPAD" text on the top border
- Confirm "QUANTIZE TRIGGER" label appears above the Off/Live/Post buttons row
- Confirm the quantize subdiv ComboBox shows "1/32" (not "1/..." truncated)

**Step 4 — Verify gamepad controller name (UI-02):**
- Without a controller: confirm label shows "No controller" in grey
- Connect a PS4/PS5/Xbox controller: confirm label updates to the correct type string in green ("PS4 Connected", "PS5 Connected", or "Xbox Connected")
- If no controller available: confirm "No controller" is shown (grey, not green)

**Step 5 — Verify looper position bar (UI-03):**
- Press PLAY on the looper (with a loop already recorded, or just start empty playback)
- Confirm a thin green bar sweeps from left to right inside the LOOPER panel
- Let it loop around — confirm the bar smoothly resets to the left edge (no backward jump)
- Press STOP — confirm the bar disappears (empty/no fill)
- Press REC — confirm the bar sweeps during recording the same as during playback
- Set a 4-bar loop at 180 BPM — confirm the sweep is continuous (no jerky updates)

**Step 6 — Verify installer (UI-04):**
Compile the installer:
```
"C:/Program Files (x86)/Inno Setup 6/ISCC.exe" installer/DimaChordJoystick-Setup.iss
```
Run `installer/Output/DimaChordJoystickMK2-Setup.exe` — on the installer welcome/information page, confirm the version string shows 1.3 (visible in the title bar or About section).
