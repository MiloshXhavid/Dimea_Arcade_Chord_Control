## Task 3 — UAT checkpoint (human)

Open the plugin in the DAW. Verify each criterion:

**SC-1: Velocity-sensitive drag**
- Drag any in-scope rotary knob (e.g., transpose, population, free BPM) very slowly
  — should feel precise, roughly 300 px for a full sweep
- Drag the same knob quickly in one stroke — should sweep ~3× faster
- Change direction mid-drag — no jump; multiplier resets smoothly through zero speed
- Hold Shift while dragging — fine-tune mode, much slower than normal slow drag

**SC-2: EMA smoothing**
- Drag the LFO rate fader (linear) slowly then quickly — no stepping or sudden jump
  in speed; acceleration/deceleration feels smooth (EMA ramp)
- Same test on the Probability knob and the free BPM knob — no discrete steps visible
  in the value display while dragging

**SC-3: Hover highlight ring**
- Move mouse over any rotary knob — a faint pink-red ring appears around it
- Move mouse away — ring disappears immediately
- Verify on a small knob (e.g., octave offset) and a large knob (e.g., free BPM)

**SC-4: 12-dot indicators on octave / interval / transpose knobs**
- Open plugin, look at the 8 target knobs: Transpose, Third/Fifth/Tension Interval,
  Root/Third/Fifth/Tension Octave
- Each should display 12 small dots evenly spaced around the arc (no fill arc visible)
- The dot at the current value is bright pink-red; the others are dim blue
- Change the value — the bright dot updates to the new position
- Confirm the red position ring is gone (replaced by the brighter dot)

Record pass/fail for each criterion. Only mark this plan complete when all four pass.
