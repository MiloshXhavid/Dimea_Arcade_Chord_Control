## Key Accomplishments

- **Resizable UI + Mini/Maxi modes** — plugin window scales 0.75×–1.0× with locked aspect ratio; 3-state toggle (Full / Mini pad-only / Maxi pad fills display); gamepad Option modes 3/4 cycle states; overlay return button; scale preserved across sessions
- **Living Space starfield** — stars drift in a unified heading accumulating from joystick input (full 360°, no snap-back); 3 parallax depth layers (100%/60%/30%); per-star twinkle ±10%; shooting star every ~20s; 2–3 soft nebula blobs; background hue shifts ±8° over 120s; INV co-rotates background photo and starfield as one unit; all effects crossfade out as warp ramps in
- **Pitch Axis Crosshair + Smart Chord Display** — two subtle lines from cursor to pad edges show root/third (horizontal) and fifth/tension (vertical) note names in real time; right-click toggle persisted in APVTS; chord name always inferred root-up (Cm7, Gmaj7#11, D6, sus2/sus4) with third inferred from active scale when Voice 1 not triggered; display holds last name during silence
- **Velocity-sensitive knob drag + 12-dot indicators** — slow drag (<2 px/frame) = fine, fast drag (>10 px/frame) = 3× sweep; EMA smoothing eliminates zipper noise; hover highlight ring; all 8 octave/interval buttons show 12 evenly-spaced position dots with current value brighter
- **Custom CC routing** — LFO CC Dest and Left Stick X/Y Mode dropdowns include "Custom CC…" option; any CC 0–127 accepted; custom assignments persist in presets; 4 inline text label members; INV swap extended to cover custom entries
- **Arpeggiator 8-step pattern grid** — ON/TIE/OFF per cell, 2-row×4-col grid in 84px arp box; LEN combo (1–8 active steps); 3-state RND SYNC button (FREE/INT/DAW); post-UAT correctness fixes included
- **Sister LFO Attenuation** — lfoXSisterAtten/lfoYSisterAtten APVTS params (−1…+1); inline bipolar slider in Sister row when cross-mod target selected; "None" = full-width combo, slider hidden
- **Modulation polish + CC routing expansion** — MOD FIX dispatch fixed for 8 missing LFO target cases; filter CC routing expanded from 4 to 18 named CCs; MOD FIX Freq target now additive over Rate slider base value
- **Rec lane undo + stability** — pressing a lit Rec button (Gates/Joy/Mod Whl) clears that lane individually; play button no longer flashes when DAW Sync armed but transport stopped; JoystickPad sticky offset model with spring-damper suppression during looper playback
