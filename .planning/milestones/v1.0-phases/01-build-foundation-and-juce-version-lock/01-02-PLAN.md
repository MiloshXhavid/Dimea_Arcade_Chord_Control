  <how-to-verify>
**Step 1 — Rescan VST3 plugins in Ableton**
1. Open Ableton Live 11.
2. Go to Preferences → Plug-Ins and click "Rescan" (or restart Ableton if you rescanned recently).
3. In the browser under Plug-ins → VST3, look for "ChordJoystick".
   - PASS: ChordJoystick appears in the list.
   - FAIL: Plugin not visible — report the exact Ableton error, if any. Check that C:\Program Files\Common Files\VST3\ChordJoystick.vst3\ exists on disk.

**Step 2 — Load the plugin on a MIDI track**
1. Create a new MIDI track (Cmd+Shift+T or via the Add menu).
2. Drag ChordJoystick from the browser onto the MIDI track.
   - PASS: Plugin editor opens showing the joystick pad, touch-plates, knobs.
   - FAIL: "Plugin could not be loaded" error — note the exact error text and report it. If Ableton shows "The plug-in … reported an error", try IS_SYNTH TRUE as a follow-up (see RESEARCH.md open question 1).

**Step 3 — Route to a soft synth and trigger a note**
1. Create a second MIDI track.
2. Add Ableton's Wavetable (or any instrument) to the second track.
3. On the second track: set "MIDI From" to "ChordJoystick" (the first track).
4. Set the channel dropdown to "ChordJoystick" (the plugin's MIDI output).
5. Set Monitor to "In".
6. Make sure the second track is armed or Monitor is on.
7. In ChordJoystick's editor, click and hold the ROOT touch-plate button.
   - PASS: You hear a note from Wavetable.
   - PARTIAL: You see gate activity (touch-plate lights up red) but hear no sound — check the routing (channel selector on the second track, Monitor setting).
   - NOTE: Ableton Live 11 merges all MIDI channels to channel 1 when routing between tracks. The 4-voice multi-channel output will appear on one channel — this is an Ableton architectural constraint, not a plugin bug. A single note sounding from any voice passes this test.

**Step 4 — APVTS round-trip test**
1. In ChordJoystick's editor, change these parameters to non-default values:
   - Global Transpose knob: set to +5
   - 3rd Interval knob: set to 3
   - Scale Preset dropdown: set to "Dorian" (or any non-default preset)
2. Save the Ableton project (Cmd+S).
3. Close Ableton completely.
4. Reopen the saved project.
5. Open the ChordJoystick editor.
   - PASS: Global Transpose = 5, Third Interval = 3, Scale Preset = Dorian (the values you set).
   - FAIL: Parameters reset to defaults — report which parameters did not restore.

**Known acceptable limitations for this smoke test (do NOT fail on these):**
- Ableton merges MIDI channels — all 4 voices appear on channel 1. This is expected.
- LooperEngine mutex: under light load the mutex will not cause issues. Do not enable any looper features during this test.
- The joystick pad may produce no sound if no voice trigger source is active — use the touch-plates only.
  </how-to-verify>
