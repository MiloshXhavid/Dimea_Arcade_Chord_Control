  <how-to-verify>
Open your DAW (Reaper or Ableton) and load ChordJoystick on a MIDI track before a software instrument. Run these 8 tests in order:

TEST 1 - UI Layout
- Open the plugin editor.
- Confirm: 4 attenuator knobs (X Range, Y Range, Cut, Res) appear directly above the 4 touchpad columns, NOT in a side column.
- Confirm: Bars control is a horizontal linear slider (not a rotary knob), range 1-16.
- Confirm: In the looper button row, REC GATES appears to the LEFT of REC JOY.
- Expected: All 3 layout changes visible.

TEST 2 - Scale Keyboard Active Note Color
- Select any scale preset (e.g., Major).
- Confirm: Active scale notes on the keyboard at the bottom of the editor highlight RED, not teal/cyan.
- Expected: Red highlights on active notes.

TEST 3 - REC TOUCH Button
- Click [REC TOUCH]. The button should indicate armed/waiting state.
- Do NOT play a note. Wait 2 seconds.
- Click [REC TOUCH] again. The wait should cancel (button returns to idle).
- Click [REC TOUCH] again, then immediately play a note on a MIDI keyboard or trigger a voice.
- Expected: Recording starts on the note-on, not immediately on button press.

TEST 4 - Looper Auto-Stop
- Set Bars to 2 and Time Sig to 4/4 (= 2-bar loop).
- Start DAW playback. Click [REC GATES] or [REC JOY] to begin recording.
- Play some notes and wait.
- Expected: Recording stops automatically after 2 bars without pressing REC again. The looper enters playback.

TEST 5 - Gamepad Button Remap
- Connect a gamepad (PS4 or Xbox controller).
- With a soft synth on the next MIDI track, press each shoulder button:
  - L1 fires voice 0 (Root / channel 1)
  - L2 fires voice 1 (Third / channel 2)
  - R1 fires voice 2 (Fifth / channel 3)
  - R2 fires voice 3 (Tension / channel 4)
- Monitor the MIDI output in the DAW MIDI monitor.
- Expected: Each button fires the correct voice. (Prior mapping was R1=Root, R2=Third, L1=Fifth, L2=Tension. Confirm the NEW mapping is active.)

TEST 6 - Filter CC DAW Visibility
- With the gamepad connected, open the DAW automation lane for the ChordJoystick plugin.
- Look for "Filter Cut CC" and "Filter Res CC" parameters in the automation parameter list.
- Move the gamepad left stick (X axis for cut, Y axis for resonance).
- Expected: Both parameters appear in the automation list AND move in real-time with left stick input.

TEST 7 - JOY Gate Hold (1-Second Stillness)
- Set at least one voice to JOY trigger source.
- Move the joystick pad into an active zone. A note should gate open.
- Slowly move to a new position on the same voice.
- Expected: Gate stays open (no note-off/note-on retrigger between positions).
- Return the joystick to center and hold still.
- Expected: Gate closes approximately 1 second after the stick returns within the dead zone (not the old 50 ms immediate close).

TEST 8 - JOY Pitch CV (Pitch Bend)
- Set at least one voice to JOY source. Move the stick to open a gate.
- While the gate is open, slowly move the stick to a new pitch position.
- Monitor the MIDI output: expected pitch bend messages (not note-off + note-on) as the pitch changes.
- Move to the pitch range extreme. Expected: pitch bends smoothly up to +-24 semitones from the gate-open pitch without a new note-on firing.
- Return to center. Confirm the gate closes after about 1 second of stillness and a pitch-bend reset to center (0x2000) is emitted on gate-close.
