## Plan 04-01 — all code tasks complete (many iterations)

Final JOY gate model (cfd547e — latest):
- Gate detection uses DELTA (movement per block), NOT absolute position
  - Absolute position caused gate to be permanently open (Y=-1.0 rest = absY=1.0 >> threshold)
- Per-axis per-voice:
  - Voices 0 (Root) + 1 (Third) → Y axis delta only
  - Voices 2 (Fifth) + 3 (Tension) → X axis delta only
- Gate opens: axisDelta > joystickThreshold
- While open + moving: pitch tracks scale quantizer
  - noteOff(oldPitch) → noteOn(newPitch) when pitch changes (works with any synth)
  - 30ms pitch debounce (joyPendingPitch_/joyPendingSamples_) prevents jitter retriggering
- Gate closes: axisDelta < joystickThreshold * 0.15f (hysteresis) for 50ms
- Silence when joystick still ✓

<decisions_made>

- JOY gate uses delta (movement speed) not absolute position — permanent gate bug fixed
- Per-axis: Y axis for Root/Third voices, X axis for Fifth/Tension voices (matches ChordEngine mapping)
- 30ms pitch debounce prevents rapid boundary-jitter retriggering
- Hysteresis close threshold = 15% of open threshold (gate ignores brief dips mid-movement)
- 50ms stillness timer triggers gate close
- noteOff(old) → noteOn(new) for pitch change — works universally, no pitch bend needed
- Pitch bend was tried and rejected (user wanted exact scale degrees, not smooth glide)
- Legato (note-on-before-note-off) was tried and rejected (causes polyphonic overlap on non-mono synths)
- Global ALL pad fires only PAD-mode voices (JOY/RND voices skipped)
- ScaleKeyboard listener uses callAsync to dispatch repaint from audio thread to message thread
- joystickYAtten default=12 (one octave), joystickXAtten default=24 (two octaves)
