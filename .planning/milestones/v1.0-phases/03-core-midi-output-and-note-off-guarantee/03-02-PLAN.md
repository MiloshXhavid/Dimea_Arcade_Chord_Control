1. Press ROOT pad

**Test 4 — Bypass note-off flush:**
1. Hold ROOT pad (note-on visible in MIDI monitor)

4. Disengage bypass — plugin resumes normally

**Test 5 — Transport stop (no forced note-off):**
1. Hold ROOT pad while Reaper transport is playing
2. Stop transport
3. Verify: note continues to sustain (no note-off sent) — the held pad keeps the note alive until released
4. Release pad — note-off fires normally

1. In Reaper, right-click the plugin and open its parameter panel (or use DAW automation)
